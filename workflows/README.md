# Workflows

Esta pasta contém os SOPs do projeto WAT. Cada workflow deve descrever:

- objetivo;
- entradas necessárias;
- ferramenta(s) determinística(s) a executar;
- saída esperada;
- tratamento de exceções.

Workflows disponíveis:

- `00_overview.md`: arquitetura WAT e fluxo geral;
- `01_setup.md`: configuração local;
- `02_ingest_documents.md`: ingestão multimodal;
- `03_query_rag.md`: recuperação e geração;
- `04_chat_ui.md`: operação da interface;
- `05_testing.md`: validações automatizadas e manuais.
